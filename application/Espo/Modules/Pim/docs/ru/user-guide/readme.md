# TreoPIM User-guide

**TreoPIM** - система предназначенная для управления товарами и продуктами, сбора товарных данных из разных источников, управление товарами, каталогами, каналами дистрибуции и ценооборазованием для различных каналов, сохранения информации о заказах, импорт и экспорт данных о товарах из и для различных площадок продаж.
